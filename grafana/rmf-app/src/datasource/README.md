# IBM RMF Data Source

A Grafana data source plugin for IBM® Resource Measurement Facility (RMF™) for z/OS®. This plugin connects to the RMF Distributed Data Server (DDS) to retrieve z/OS performance metrics and display them in Grafana dashboards.

This data source is bundled as part of the [IBM RMF plugin](https://github.com/IBM/RMF) (`ibm-rmf-app`).
